Bespoke Design: Creating A Tailored Digital Gift and Gift Giving Experience

Skills: Research, Design, Wireframing, Prototyping, Testing, Creativity, Narrative

Through out this project I have used a broad range of skills to create a personalized gift and gift giving experience for the assigned client.

The process started by conducting research to learn more about the user and her needs. The research consisted of a semi-structred interview, a enthographic interview, a desk research, a personal inventory and an observation. Not only did the research help defining elements that create a great gift giving experience in general, but it hepled define the elements that would create a tailored experience for the client the expeirence was being designed for.

The image below shows a pyschical prototype that was tested by the client in order to get insights on how she experiences gift giving. The information gathered here was also used in creating a user portrait which acted as reference for designing the experience.

Using all the data gathered and analysed in the research phase, I started to ideate. This held in that I would come with as many ideas as possible and then converge them to select a few.
